# kwai-video
快手短视频在线，内置视频（有能力的可以自己添加）

说明:</br>
1.简易播放器，自定连续播放。</br>
2.不支持API调用（如果要调用可以使用 api.php）</br>
3.视频源存储第三方，内置了200条左右；（有我有时间会维护，不会失效）</br>
4.需要添加视频源的在ks.txt里面。</br>
